API EXPRESS - CONTINENTES

Bienvenido a la API de Datos de Continentes

Esta API proporciona información sobre países de los continentes de América y África.

Autores: Martín Reser y Lucas Moltedo

API que utilizamos de base: Documentación de la API original

Versión API: v1.0.0

ENDPOINTS ÁFRICA

GET /api/v1/countries/africa/all

- Devuelve todos los países de África.

GET /api/v1/countries/africa/{ID}

- Devuelve un determinado país en base a un ID.

GET /api/v1/countries/africa/population?population={numero}

- Devuelve los países de África que tengan hasta {numero} habitantes.

ENDPOINTS AMÉRICA

GET /api/v1/countries/america/all

- Devuelve todos los países de América.

GET /api/v1/countries/america/{cioc}

- Devuelve un determinado país en base a su abreviación.

GET /api/v1/countries/america/language?language={lenguaje}

- Devuelve los países de América que tengan {lenguaje} como lenguaje.
